# My dotfiles

## Usage
```sh
git clone https://github.com/yoshiyoshiharu/dotfiles
bash dotfiles/setup.sh
```

## Requires
```sh
git clone https://github.com/zsh-users/antigen.git ~/.zsh/antigen/
brew install node
brew install ripgrep
brew install tmux
```

## TreeSitters
If you need erb treesitter, run below.

```vim
:TSInstall erb
```
